# 📖 关于项目

## 🎯 项目简介

这是一个基于 **Eino ADK 框架** 开发的智能狼人杀游戏对话记录展示系统。通过大语言模型驱动的AI角色扮演，记录和展示完整的狼人杀游戏过程。

## 🌟 核心特色

### 🤖 AI智能角色扮演
- 每个角色都有独特的性格特征和思维模式
- 基于大语言模型实现智能对话生成
- 角色间的互动自然流畅，符合狼人杀游戏逻辑

### 📊 完整数据记录
- 详细记录游戏的每个环节和对话内容
- 提供多维度的统计分析和数据展示
- 支持历史对局回顾和对比分析

### 🎨 现代化界面
- 基于 VitePress 构建，性能优异
- 响应式设计，适配各种设备
- 清晰的视觉层次和用户体验

## 🛠️ 技术架构

### 前端技术
- **VitePress**：静态站点生成器
- **Vue.js**：前端框架
- **Markdown**：内容管理
- **CSS3**：样式设计

### 后端技术
- **Eino ADK**：AI开发框架
- **大语言模型**：角色对话生成
- **Python**：游戏逻辑处理

## 🎮 游戏机制

### 角色配置
- **狼人** (3人)：夜间讨论，协同作战
- **村民** (3人)：白天推理，投票决策
- **预言家** (1人)：夜间查验，获取信息
- **女巫** (1人)：拥有解药和毒药
- **猎人** (1人)：淘汰时可带走一名玩家

### 游戏流程
1. **准备阶段**：角色分配，身份确认
2. **夜晚阶段**：各角色按顺序行动
3. **白天阶段**：讨论发言，投票淘汰
4. **胜负判定**：达到胜利条件即结束
