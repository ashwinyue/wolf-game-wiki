# 🐺 狼人杀游戏 Wiki - 项目概览

## 🎯 项目简介

这是一个基于 **Eino ADK 框架** 开发的智能狼人杀游戏对话记录展示系统。通过大语言模型驱动的AI角色扮演，记录和展示完整的狼人杀游戏过程。

## 🌟 核心功能

### 📖 内容模块
- **首页展示** - 项目介绍和快速导航
- **快速开始** - 新手指南和使用说明
- **游戏记录** - 详细的对局对话记录
- **角色介绍** - 各角色技能和策略说明
- **数据统计** - 游戏统计和分析报告
- **关于项目** - 项目详情和技术架构

### 🎨 设计特色
- **响应式设计** - 适配各种设备屏幕
- **现代化界面** - 清晰美观的视觉设计
- **主题定制** - 自定义CSS样式和图标
- **交互友好** - 直观的导航和布局

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 本地开发
```bash
npm run docs:dev
```

### 构建生产版本
```bash
npm run docs:build
```

### 预览构建结果
```bash
npm run docs:preview
```
